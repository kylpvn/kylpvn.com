🔋 Key Features


3D Customizable Hero Section: Features a modifiable 3D desktop model.

Dynamic Experience and Work Sections: Engages users with animations from framer motion.

3D Skills Display: Utilizes three.js and React Three Fiber to present skills in 3D.

Animated Projects and Testimonials: Brings projects and client feedback to life with framer motion animations.

Contact Section with 3D Earth Model: Incorporates a 3D model of Earth with email functionality from emailjs.

3D Stars Background: Creates a starry background with randomly positioned stars using Three.js.

Seamless Animations: Offers consistent animation effects across the site with framer motion.

Responsive Layout: Guarantees a seamless viewing experience on all devices.

Plus more, including aspects on code structure and reusability.
